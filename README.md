# API sem framework

Interface para cadastro de carros, usando modelo UI do Java para demonstração

# O que foi usado
* Derby - para persistência de dados
* JavaEE8 
* banco JDBI

Para usar basta dar um **fork** e excutar a classe ``` App ```
